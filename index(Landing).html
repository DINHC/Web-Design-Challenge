Summary

Using an Open Weather API along with Google APIs, the project shown here was made with the purpose of analyzing weather data from randomly generated cities around the world

Once we gathered the neccessary data and turning it into a csv, Pandas and Matplotlib Python were used to analyze the data set pertaining to Max Temperature, Humidity, Cloudiness, and Wind Speed. This dashboard shows the source data and the visualizations based on the four dependent variables.
